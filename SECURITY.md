# Security Policy

## Supported Versions

Only the latest release of `page-agent` is supported with security updates.

| Version | Supported          |
| ------- | ------------------ |
| >= 1.11 | :white_check_mark: |
| < 1.11  | :x:                |

## Reporting a Vulnerability

We take the security of `page-agent` seriously. If you find a security vulnerability, please do not disclose it publicly. Instead, please report it through the official Alibaba security channels:

1. **Alibaba Security Response Center (ASRC):** Submit a report at **[https://security.alibaba.com/](https://security.alibaba.com/)**.
2. **Alibaba HackerOne Program:** Submit a report via **[https://hackerone.com/alibaba](https://hackerone.com/alibaba)**.

Please include steps to reproduce, impact details, and environmental context. We will review and remediate reports promptly in accordance with ASRC policies.
